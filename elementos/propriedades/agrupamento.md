# Agrupamento
{: .no_toc }

## Sumário
{: .no_toc .text-delta }

1. TOC
{:toc}

Todos os elementos (exceto o desenho com a caneta) podem ser agrupados. Essa ação permite mover os elementos conjuntamente e não mescla formas. As opções de agrupamento estão disponíveis ao selecionar múltiplos elementos e selecionar o ícone de engrenagem <img src="/assets/engrenagem.png" width="18"> disponível na barra direita do projeto.

## Agrupar elementos
O agrupamento <img src="/assets/agrupar.png" width="18"> agrupa os elementos de forma que possam ser movidos em conjunto com o botão esquerdo do mouse. Essa ação pode ser desfeita.

## Desagrupar elementos
O desagrupamento <img src="/assets/desagrupar.png" width="18"> desagrupa elementos previamente agrupados de forma que possam ser movidos separadamente com o botão esquerdo do mouse.
