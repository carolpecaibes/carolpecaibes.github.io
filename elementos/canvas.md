# Canvas
O canvas é o grid infinito no qual todos os elementos do projeto são incluídos. Para mover a visualização do canvas basta clicar com o botão direito do mouse em qualquer parte vazia do grid e arrastá-lo.

Para rolar a visualização do canvas verticalmente deve-se usar a roda de rolagem do mouse. Para rolar horizontalmente deve-se pressionar a tecla `Shift` enquanto usa a roda de rolagem do mouse.
