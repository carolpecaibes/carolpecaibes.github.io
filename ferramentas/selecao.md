# Seleção
A ferramenta de seleção <img src="/assets/selecao.png" width="18"> está localizada na barra esquerda do projeto. Ela serve para selecionar elementos no canvas como formas, textos, objetos e atores. Para utilizá-la, deve-se usar o botão esquerdo do mouse.
