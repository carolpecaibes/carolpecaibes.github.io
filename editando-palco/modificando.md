# Modificando o palco

Para modificar um palco criado é necessário destravá-lo:
1. Selecione a ferramenta de camadas <img src="/assets/camadas.png" width="18"> disponível na barra direita do projeto
2. Clique no ícone do cadeado fechado <img src="/assets/cadeadofechado.png" width="18">, que mudará para um cadeado aberto <img src="/assets/cadeadoaberto.png" width="18">
3. Altere as características desejadas
4. Clique no ícone do cadeado aberto <img src="/assets/cadeadoaberto.png" width="18"> para travar novamente o palco
